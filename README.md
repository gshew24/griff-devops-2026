# FitTrack

FitTrack is a simple nutrition and macro tracking web application.

## Technology Stack

- Node.js
- Express.js
- MongoDB Atlas
- RESTful API
- Bootstrap / CSS
- GitHub Actions CI/CD
- Google Cloud Platform VM
- NGINX reverse proxy
- HTTPS via Let's Encrypt

## Features

- Track daily food entries
- Calculate calories, protein, carbs, and fat
- Full CRUD operations
- Persistent MongoDB-backed data
- Responsive single-page application
